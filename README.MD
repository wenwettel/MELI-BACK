# Meli Server

## Description

Api that provides information regarding items for a particular search criteria or for a particular ID

## Tecnologies

- Express
- Node
- Axios
- DotEnv
- CORS

## Installation Guide

### Prerequisites

- Node version: "16.14.0"
- Default PORT 3001

### Install

- npm install

### Run app

- npm run dev
